# Stopwatch-radio-driver-for-Glider-rc-contest

This is a small portable device connecting to the displays compatible with Osegouin design https://github.com/osegouin/DisplayGliderScore/
It is particullary usefull with https://github.com/f5mmx/Gliderscore-portable-display-and-stopwatch when a team is attending to a RC contest using a different display driver.

Instead of having a large and bright display showing the working time computed by GliderScore, each user can have a small device that will show the Group number, the Round number, the official working time and have a stopwatch.
